# Учебный проект

Этот проект создан исключительно в учебных целях.  

**Важно:**  
- Сайт не имеет реального функционала.  
- Нет подключения к серверу, базе данных или внешним API.  
- Все данные - статичные, любые действия на сайте не сохраняются и не обрабатываются.  

Проект использовался для практики вёрстки/фронтенда (*HTML/CSS/JS*).  

Реальный бэкенд отсутствует. Любые формы, кнопки и интерактивы - только имитация.

# Educational Project

This project was created solely for learning purposes.

**Note:**  
- The site has no real functionality.  
- No server connection, database, or external API.  
- All data is static - any actions on the site are not saved or processed.
